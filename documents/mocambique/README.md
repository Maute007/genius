# Documentos Moçambicanos

Este diretório é para os documentos que serão processados pelo sistema RAG do Genius.

## Como usar:

1. Coloque aqui os seus 100 documentos moçambicanos (PDF, Word, TXT)
2. Os documentos serão processados automaticamente
3. O conteúdo será usado para enriquecer as respostas da IA com contexto local

## Formatos suportados:
- PDF (.pdf)
- Word (.docx, .doc)
- Texto (.txt)
- Markdown (.md)

## Estrutura recomendada:
- curriculo/ - Documentos do currículo nacional
- materias/ - Material didático por matéria
- exemplos/ - Exemplos e exercícios moçambicanos
- contexto/ - Informações culturais e locais

Após adicionar os documentos, o sistema irá processá-los e criar embeddings para busca semântica.
